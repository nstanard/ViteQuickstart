# Vite Quick Start

A simple Vite quick start project that has all the core tools to be successful.

## Features & Packages

- TypeScript
- ESLint & Prettier
- Cypress & Jest w/ coverage reporting
- Table and Query tanstack packages
- React router v6
- Normalize
- Docker

## What's missing?

- Re-usable components/styling
